# #33 더보기 설정 화면 분리 — 우측 상단 톱니바퀴 버튼

## 개요

더보기 화면에 기능 메뉴(예산, 정기지출 등)와 설정(테마, 알림)이 혼재되어 있다.
우측 상단 톱니바퀴(⚙️) 버튼으로 `SettingsScreen`을 분리하여 역할을 명확히 구분한다.

| 화면 | 성격 | 항목 |
|------|------|------|
| **더보기** | 앱 기능 허브 | 재무상태, 예산, 정기지출, 저축률 목표, 자산 목표, 가족 정보 |
| **설정** | 계정·환경 설정 | 화면 모드, 알림 설정, 로그아웃, 회원 탈퇴 |

---

## 화면 구성

### 더보기 (MoreMenuScreen) — 메뉴 재정렬

우선순위 기준: 사용 빈도 높음 → 낮음 순, 계정 관련은 설정으로 이전

```
┌─────────────────────────────────┐
│  더보기               [⚙️]      │  ← 우측 상단 톱니바퀴
├─────────────────────────────────┤
│  [프로필 카드]                   │
├─────────────────────────────────┤
│  재무상태             >          │  1순위 — 매월 사용
│  예산 설정            >          │  2순위 — 자주 확인
│  정기 지출 관리       >          │  3순위 — 고정비 관리
│  저축률 목표 설정     >          │  4순위 — 목표 설정
│  자산 목표 설정       >          │  5순위 — 목표 설정
│  가족 정보            >          │  6순위 — 가끔 확인 (#31)
└─────────────────────────────────┘
```

**이전한 항목:** 화면 모드, 알림 설정, 로그아웃 → SettingsScreen

### 설정 (SettingsScreen) — 신규

톱니바퀴 탭 시 → `SettingsScreen` navigate

```
┌─────────────────────────────────┐
│  ← 설정                          │
├─────────────────────────────────┤
│  환경                            │  (섹션 헤더)
│  화면 모드           시스템 >    │
│  알림 설정                  >    │
├─────────────────────────────────┤
│  계정                            │  (섹션 헤더)
│  로그아웃                        │  danger 색상
│  회원 탈퇴                        │  textTertiary 색상 (#30)
└─────────────────────────────────┘
```
