# 기획 — 추가 기능 목록

> 작성일: 2026-03-06

---

## 1. 예산 설정 & 달성률

**개요**
카테고리별로 월 예산을 설정하고, 실제 지출 대비 달성률을 시각화한다.

**주요 기능**
- 카테고리별 월 예산 금액 설정
- 홈/통계 화면에 예산 대비 소비율 진행 바 표시
- 예산 80% 초과 시 경고, 100% 초과 시 알림

**데이터 모델 (Firestore)**
```
families/{familyId}/budgets/{yearMonth}/categories/{categoryKey}
  - amount: number
```

**우선순위**: 높음 — 가계부 앱의 가장 실용적인 핵심 기능

---

## 2. 정기 지출 (고정비) 관리

**개요**
매달 반복되는 거래(월세, 구독, 보험 등)를 등록하고 자동으로 이번 달 거래에 추가한다.

**주요 기능**
- 반복 거래 등록 (매월 특정 일자, 금액, 카테고리)
- 월 초에 고정비 자동 생성 또는 확인 알림
- 이번 달 아직 반영 안 된 고정비 목록 표시

**데이터 모델 (Firestore)**
```
families/{familyId}/recurringTransactions/{id}
  - title: string
  - amount: number
  - category: string
  - dayOfMonth: number
  - type: 'expense' | 'income'
  - isActive: boolean
```

**우선순위**: 높음 — 가족 단위 앱에서 필수적인 기능

## 3. 가족 멤버별 지출 분리

**개요**
거래 등록 시 어떤 가족 구성원의 지출인지 태깅하고, 멤버별로 통계를 확인한다.

**주요 기능**
- 거래 추가 시 멤버 선택 (기본값: 나)
- 통계 화면에서 멤버별 지출 breakdown
- 홈 화면 메트릭에 멤버별 이번달 소비 표시

**데이터 모델 변경**
```
transactions/{id}
  - memberId: string  // 추가
```

**우선순위**: 높음 — 가족 앱 정체성을 살리는 핵심 차별점

## 4. 전월 비교 인사이트

**개요**
숫자만 나열하지 않고, 자동으로 생성된 텍스트 인사이트로 지출 패턴을 해석해준다.

**주요 기능**
- "지난달보다 식비가 15만원 더 나왔어요"
- "이번달 저축률이 가장 높은 달이에요"
- 통계 화면 상단에 인사이트 카드 표시

**구현 방식**
- `monthlySummaries`의 전월 데이터와 비교하여 클라이언트에서 텍스트 생성
- 별도 백엔드 불필요

**우선순위**: 중간 — UX 완성도를 높이는 기능

---

## 5. 연간 요약

**개요**
월 단위에 집중된 현재 통계를 보완하여 연간 흐름을 한눈에 파악한다.

**주요 기능**
- 올해 총 수입 / 총 지출 / 총 저축액
- 월별 수입 vs 지출 나란히 보여주는 막대그래프 (12개월)
- 연간 저축률 표시

**구현 방식**
- `monthlySummaries`에서 해당 연도 데이터 일괄 조회
- 통계 화면에 연/월 탭 추가

**우선순위**: 중간

---

## 6. 저축률 트래킹

**개요**
매달 저축률을 계산하고 목표 달성 여부를 추적한다.

**주요 기능**
- 저축률 = (수입 - 지출) / 수입 × 100
- 목표 저축률 설정
- 홈 화면에 이번달 저축률 표시 및 목표 대비 달성 여부

**우선순위**: 중간

---

## 7. 거래 검색 & 필터

**개요**
거래 목록에서 원하는 거래를 빠르게 찾을 수 있다.

**주요 기능**
- 메모, 카테고리, 금액 범위로 검색
- 날짜 범위 필터
- `TransactionListScreen` 상단에 검색 바 추가

**우선순위**: 중간 — 거래가 쌓일수록 필요성 증가

---

## 8. 거래 사진 첨부 (영수증)

**개요**
거래에 영수증 사진을 첨부하여 근거를 남긴다.

**주요 기능**
- 거래 추가/수정 시 사진 첨부 (카메라 또는 갤러리)
- Firebase Storage에 이미지 저장
- 거래 상세 화면에서 첨부 사진 확인

**데이터 모델 변경**
```
transactions/{id}
  - receiptUrl?: string  // 추가
```

**우선순위**: 낮음 — 편의 기능

---

## 9. 자산 목표 설정

**개요**
"1억 모으기" 같은 목표 금액을 설정하고 달성 진행률을 홈 화면에서 확인한다.

**주요 기능**
- 목표 금액 및 목표명 설정
- 홈 화면 실자산 카드에 목표까지 남은 금액 + 진행 바 표시
- 목표 달성 시 축하 애니메이션

**데이터 모델 (Firestore)**
```
families/{familyId}/goals/{id}
  - title: string
  - targetAmount: number
  - createdAt: timestamp
```

**우선순위**: 낮음 — 장기 동기부여 기능

---

## 10. 푸시 알림

**개요**
중요한 시점에 알림을 보내 앱 활용도를 높인다.

**알림 종류**
- 월 초: "이번 달 예산을 설정해보세요"
- 예산 80% 초과: "식비 예산의 80%를 사용했어요"
- 고정비 등록일: "월세 납부일이에요"

**구현 방식**
- Firebase Cloud Messaging (FCM)
- Cloud Functions로 스케줄 트리거

**우선순위**: 낮음 — 인프라 작업 필요
